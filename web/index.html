Linux运维派开源镜像站

Linux运维派开源镜像站是一个以普及开源软件，方便开源爱好者和用户高效访问开源项目的各种资源的非盈利计划。

该镜像站提供了包括 Arch Linux, CentOS, CPAN, CTAN, Cygwin, Debian, Deepin, EPEL, Fedora, Kali Linux, Linux Mint, openSUSE, php，nginx，zabbix, Ubuntu 等项目源的镜像，以服务于教育和科学研究为目的，提倡自由、平等、协作、共享的精神。
